# studio-web

Web frontend interface for Quatrain Studio.

## Purpose

This package is part of the Quatrain Core monorepo. It provides a visual dashboard and user interface for interacting with the Quatrain Studio platform.

## HOWTO / Usage Examples

```bash
# Start the development server
yarn dev

# The interface will be accessible at
# http://localhost:3000
```

## Conventions & Technical Details

- Follows the standard Quatrain architecture patterns.
- Ensure proper configuration before usage.
- See the root monorepo documentation for more details on building and testing this package.
